Keystroke Explanations

Keystroke: F5

repaint / refresh the window

Keystroke: F11

Fullscreen

Keystroke: SPACE (Playlist-Mode only)

next slide

Keystroke: ARROW LEFT / RIGHT (Playlist-Mode only)

previous / next slide

Keystroke: END / PAUSE (Playlist-Mode only)

blank slide

Keystroke: Ctrl + (Number) (Playlist-Mode only)

Keep Ctrl pressed and type numbers -> number x.
When you release Ctrl you change to slide x
e.g. (Ctrl) + 3 + 4 -> slide 34
